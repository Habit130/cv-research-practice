# 科研指南工作区

计算机视觉科研训练工作区:第一阶段产出理论指南,第二阶段围绕一个小型 CV 任务真实执行实践闭环。本文件只做词汇表,不放实现细节。

## Language

**理论指南**:
第一阶段的成果,即根目录的《计算机视觉科研入门指南.pdf》,回答"科研训练要求什么、如何建立能力地图"。
_Avoid_: 入门指南、第一份指南(指代不清时用全名)

**实践闭环**:
第二阶段的本体——围绕一个小型 CV 任务真实跑完"论文选读 → 代码复现 → 实验记录 → 对比分析 → 报告写作"五个环节。交付物是闭环本身及其记录,外加一份 README 式使用文档;不产出第二份 PDF 指南。
_Avoid_: 实践指南(易被误解为要写第二份指南文档)

**使用文档**:
面向科研初学者(与理论指南同一读者)的 README,教读者围绕自己的任务跑一遍实践闭环,使本仓库成为可克隆的实践模板。
_Avoid_: 说明书、手册

**验证跑**:
作者本人对全流程的一次初步执行,目的是验证使用文档的每一步可被陌生人照做,而非产出科研成果。
_Avoid_: 我的实验(与读者自己的闭环运行混淆)

**任务梯度**:
使用文档中"如何选任务"的三档参考:MNIST(最低档,验证环境、纯 CPU 可跑)→ CIFAR-10 分类(验证跑示例)→ 小型目标检测(理论指南贯穿例子,进阶自选)。MNIST 同时充当 Phase 0 的冒烟测试,不单独走闭环。
_Avoid_: 难度分级、任务列表
