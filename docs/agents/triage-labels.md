# Triage 标签

> 默认脚手架(2026-07-07):五个角色均使用默认字符串,未做任何映射;待讨论后按需覆盖右列。

技能以五个规范 triage 角色来表述状态。本文件把这些角色映射到本仓库 tracker 实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库 tracker 中的标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已完整规格化,可交给 AFK agent 执行 |
| `ready-for-human` | `ready-for-human` | 需要人来实现 |
| `wontfix` | `wontfix` | 不予处理 |

当技能提到某个角色(例如"apply the AFK-ready triage label")时,使用表中对应的标签字符串。

如需换用自己的词汇,直接编辑右列即可。
