# 0002 仓库只含流程文本,永不含任务代码与数据

本仓库(cv-research-practice)只包含流程文件:README、模板、验证跑的笔记/记录/报告、维护者层。任务代码来自读者(和作者)各自克隆的上游仓库,数据集与 checkpoint 留在上游仓库自己的数据目录。理由:模板的生命力在流程而非某份代码快照;一旦收录代码,仓库就要跟着上游演化背包袱,克隆体积也失控——纯文本仓库保持在几十 KB。

## Consequences

- 验证跑的复现对象克隆到仓库**外**(或 `.gitignore` 掉),实验记录里用 URL + commit hash 指认。
- README 的硬件声明只描述"在什么设备、什么路径下跑",不搬运任何训练脚本。
